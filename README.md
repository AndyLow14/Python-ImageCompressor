# Python-ImageCompressor
A python program that uses Pillow to compress images in batches.

1. Create a **ToCompress** and **Compressed** Folder [Case sensitive]
2. Place all images that you want to compress in the **ToCompress** folder
3. Run PyImageCompressor.py with python ./PyImageCompressor.py
4. Enter your final image quality [1 for least quality | 60 for best quality]
5. All compressed images will be saved at the **Compressed** folder
